# 복습 알림 메시지 개선 디자인

**날짜:** 2026-04-11
**상태:** 기획중

---

## 배경

현재 복습 알림 메시지는 톤이 약하고 아침/저녁 구분이 없어 클릭 유인이 낮다.
수능을 준비하는 고등학생 대상으로 긴박감을 높인 문구로 교체한다.

---

## 결정 사항

### 1. 알림 발송 구조 변경

**Before:** 미완료 과제마다 개별 알림 → 과제 3개면 아침 알림 3개 발송

**After:** 시간대당 알림 1개만 발송
- 대표 과제: `scheduledFor`가 가장 이른 incomplete task 1개를 선택
- 결과: 하루 최대 알림 2개 (아침 1 + 저녁 1)

알림이 없어야 할 때: 미완료 과제가 0개면 알림 발송하지 않음 (기존 동일)

### 2. 알림 문구

#### 아침 7:30

| 조건 | 타이틀 | 바디 |
|------|--------|------|
| 약점 라벨 있음 | `벌써 잊혀지고 있어요. {약점}, 지금 3분이면 돼요` | `오늘 안 하면 내일 처음부터예요` |
| 약점 라벨 없음 (예외) | `벌써 잊혀지고 있어요. 지금 3분이면 돼요` | `오늘 안 하면 내일 처음부터예요` |

#### 저녁 20:00

| 조건 | 타이틀 | 바디 |
|------|--------|------|
| 약점 라벨 있음 | `{약점}, 오늘 자기 전 마지막 기회예요` | `잠들기 전 3분, 기억이 굳어져요` |
| 약점 라벨 없음 (예외) | `오늘 복습 마감, 자기 전 3분만요` | `잠들기 전 3분, 기억이 굳어져요` |

`{약점}` = `diagnosisMap[task.weaknessId].labelKo` (예: `판별식 계산 실수`)

### 3. 톤 선택 근거

- **긴박감 A-3** 선택: 수능 준비생 대상, 직접적인 압박이 동기부여에 효과적
- **현재진행형 오프닝** (`벌써 잊혀지고 있어요`): 에빙하우스 망각 곡선 맥락, 지금 이 순간 뭔가 일어나고 있다는 느낌
- **아침/저녁 다른 바디**: 아침은 "처음부터" 위협, 저녁은 "기억이 굳어진다" 보상 프레이밍

## 구현 범위

변경 파일: `features/quiz/notifications/review-notification-scheduler.ts`

- `scheduleReviewNotifications`: 루프 → 대표 과제 1개 선택 + 아침/저녁 문구 분리
- `rescheduleAllReviewNotifications`: 변경 없음 (내부적으로 위 함수 호출)
- notification identifier: `review_morning`, `review_evening` (task별 ID → 고정 ID로 단순화)

---

## 테스트 기준

- 미완료 과제 0개: 알림 스케줄 없음
- 미완료 과제 1개: 아침/저녁 각 1개, 약점 라벨 포함
- 미완료 과제 3개: 아침/저녁 각 1개 (가장 오래된 과제 기준), 나머지 과제는 알림 없음
- 약점 라벨 없는 예외: 일반 문구로 대체
