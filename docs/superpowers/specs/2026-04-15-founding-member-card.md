## 한 줄 요약

설정 탭 상단에 골드 테두리 카드로 Founding Member 혜택을 표시한다.
로그인한 유저에게만 보이며, 항상 고정된다.

---

## 배경

유료화 전략(2026-04-14-monetization-strategy.md)의 핵심 약속:
- **지금 사용하는 기능은 영원히 무료** — 기존 유저는 아무것도 잃지 않는다
- **앞으로 출시될 프리미엄 기능은 1년 무료** — Founding Member 특전

이 카드는 그 약속을 앱 안에서 명시적으로 전달한다.
유저가 유료화 소식을 나중에 들어도 "이미 알고 있었다"는 신뢰 기반을 만든다.

---

## 유료화 구조 (확정)

| 구분 | 정책 |
|------|------|
| 현재 기능 (진단, 오답 분석 등) | 영구 무료 유지 |
| 앞으로 출시될 프리미엄 기능 | 유료 출시 후 Founding Member에게 1년 무료 |
| 익명/미로그인 유저 | Founding Member 혜택 없음 |

---

## 카드 스펙

### 위치
`features/profile/components/profile-screen-view.tsx`
`heroCard` 바로 아래, 에러/알림 카드보다 앞.

### 노출 조건
```ts
profile?.accountKey.startsWith('user:')
```
미인증(익명) 유저에게는 렌더링하지 않는다.

### 닫기
없음. 항상 고정. 계정 특전 배지이므로 dismiss 불필요.

### 디자인 (골드 테두리)

| 속성 | 값 |
|------|----|
| 배경색 | `#FFFBEB` (amber-50) |
| 테두리 | `2px solid #F59E0B` (amber-400) |
| border-radius | `BrandRadius.lg` (기존 카드와 동일) |
| padding | `BrandSpacing.lg` |
| 아이콘 | 🥇 |
| 제목 색 | `#92400E` (amber-800) |
| 본문 색 | `#78350F` (amber-900) |
| 구분선 | `#FDE68A` (amber-200) |
| 하단 레이블 색 | `#B45309` (amber-700) |

### 카피

**제목 행:** 🥇 Founding Member

**본문:**
> 지금 사용하시는 모든 기능은 무료로 유지됩니다.
> 앞으로 출시될 프리미엄 기능도 Founding Member에게는 1년간 무료로 제공됩니다.

**하단 구분선 아래 두 레이블:**
- 왼쪽: `현재 기능 영구 무료`
- 오른쪽: `프리미엄 1년 무료`

## 제외 범위

- 모달/토스트 없음 — 설정 카드 하나만
- 애니메이션 없음 — 정적 UI
- 서버 연동 없음 — 클라이언트 조건부 렌더링만
- Founding Member 여부를 서버에서 검증하지 않음 — 현재 단계에서 모든 로그인 유저 = Founding Member

---

## Success Criteria

- 로그인 유저: 설정 탭 열면 heroCard 바로 아래에 골드 카드 표시
- 익명 유저: 카드 표시되지 않음
- 카드 내 카피가 확정된 문구와 일치
- 기존 설정 카드들의 레이아웃에 영향 없음
