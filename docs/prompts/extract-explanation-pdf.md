# 해설 PDF 추출 프롬프트

> 코워크 세션에 아래 내용을 붙여넣어 사용하세요.
> 해설 PDF 파일을 먼저 첨부한 후 프롬프트를 전송합니다.

---

## 프롬프트 (복사해서 사용)

```
첨부한 PDF는 수능/모의고사 수학 해설지입니다.

아래 형식에 맞게 각 문제의 해설 데이터를 추출해주세요.

## 추출 규칙

- number: 문제 번호 (1~30)
- intent: [출제의도] 텍스트 그대로 추출 (예: "근호를 포함한 식의 값을 계산한다.")
- text: 풀이 서술 텍스트. 수식 기호가 깨지는 경우 [수식] 으로 표시하고 formulas에 LaTeX로 따로 기재
- formulas: 풀이에 등장하는 수식을 LaTeX 형식으로 목록 작성
  - 예: "\\sqrt{75} = 5\\sqrt{3}", "\\frac{6}{\\sqrt{3}} = 2\\sqrt{3}"
  - 수식이 없으면 빈 배열 []

## 출력 형식

반드시 아래 JSON 형식으로만 출력하세요. 설명 텍스트 없이 JSON만 출력합니다.

{
  "examId": "<파일명 기반으로 작성, 예: mock-2025-10>",
  "extractedAt": "<오늘 날짜 ISO 형식>",
  "explanations": [
    {
      "number": 1,
      "intent": "근호를 포함한 식의 값을 계산한다.",
      "text": "√75 = 5√3으로 변환하면 [수식]. 6/√3 = [수식]이므로 전체 식은 [수식]",
      "formulas": [
        "\\sqrt{75} = 5\\sqrt{3}",
        "\\frac{6}{\\sqrt{3}} = 2\\sqrt{3}"
      ]
    }
  ]
}
```

---

## 추출 후 할 일

1. 출력된 JSON을 `data/explanations/<examId>.json`에 저장
2. `npm run validate-problems <examId>` 실행으로 유효성 검사
