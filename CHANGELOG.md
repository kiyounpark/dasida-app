# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0.1] - 2026-05-07

### Added
- **개발자 도구 진입점**: 개발 빌드(`__DEV__`)에서 설정 탭 > 앱 정보 하단에 "🛠 개발자 도구" 버튼 노출, `/dev` 허브로 이동
- **복습 세션 목업**: `/dev` 허브에 `taskId=__mock__`으로 복습 세션을 바로 테스트할 수 있는 항목 추가

## [0.1.0.0] - 2026-04-27

### Added
- **진단 미니 카드**: 오답 문제 진단 완료 후 오답 패턴 요약 카드를 진단 화면에 표시. 사용자가 직접 "다음 문제" 또는 "리포트 보기"를 탭해서 진행
- **마일스톤 배너**: 오답의 1/3, 2/3 진단 시점에 격려 배너 자동 노출 (1회만 표시)
- **학습 노트 점 게이지 (NoteCollectionBar)**: 진단 완료된 문제 수를 점으로 시각화. 결과 화면과 리포트 화면에 표시
- **홈 분석 재개 카드 (ExamAnalysisResumeCard)**: 오답 진단이 진행 중일 때 홈 화면에 재개 카드 표시. 기존 여정/리뷰 카드를 대체
- **모은 노트 목록 (CollectedNotesList)**: 분석 재개 카드 아래에 지금까지 진단한 오답 패턴 목록 표시
- **홈 화면 모드 분기**: 오답 진단 진행 중이면 분석 모드, 아니면 평소 모드로 자동 전환

### Changed
- 오답 진단 화면이 자동으로 다음 문제로 넘어가던 방식 → 미니 카드 확인 후 사용자가 직접 진행하는 방식으로 변경
- 시험 결과 리포트에 클라이맥스 배너와 점 게이지 추가 (오답 있을 때만)
- 결과 화면 상단 카피 업데이트 ("학습 노트를 모아보세요" 안내 추가)

### Fixed
- 만점 시험 완료 시 이전 시험의 오답 진단 카드가 홈에 계속 표시되던 버그
- 점 네비게이션으로 비순차 이동 시 `currentNoteCountBeforeThis` 오계산 버그
- 시험 결과 리포트의 클라이맥스 배너가 실제로 표시되지 않던 버그 (props 미전달)
